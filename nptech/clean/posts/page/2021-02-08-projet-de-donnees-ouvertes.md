# Projet de données ouvertes

Le Projet de données ouvertes  est basé sur les réponses à deux enquêtes biennales : l’Enquête mondiale sur l’utilisation des technologies par les ONG, et l’Enquête sur les tendances caritatives mondiales (à paraître le 1 mars 2022). Publiées en plusieurs langues, les deux enquêtes sont 100 % anonymes et les données visent à aider les OBNL, les organisations non gouvernementales (ONG) et les organismes de bienfaisance du monde entier. Afin de voir les données, veuillez répondre à l’enquête; après avoir soumis vos réponses, vous obtiendrez instantanément un accès pour voir et explorer les données par région et par pays. Les données peuvent aussi être téléchargées dans un document Excel, en format PDF ou comme présentation PowerPoint afin de les analyser plus en profondeur.

## **Enquête mondiale 2021 sur l’utilisation des technologies par les ONG**

L’Enquête mondiale sur l’utilisation des technologies par les ONG vise à mieux comprendre comment les ONG, les OBNL et les organismes de bienfaisance du monde entier utilisent la technologie pour le marketing numérique et la collecte de fonds en ligne. Les questions de l’enquête portent sur le site Web et les communications par courriel, les outils de collecte de fonds en ligne, les médias sociaux, et les logiciels de productivité et de sécurité.

## **Comment utiliser les données**

Les données anonymisées de l’enquête peuvent facilement être téléchargées dans un fichier Excel, PowerPoint, PDF ou Word. Comme ce sont des données ouvertes, elles peuvent être utilisées et republiées par toute personne ou tout organisme, à travers le monde. Voici quelques simples suggestions :

- Les ONG, les OBNL et les organismes de bienfaisance peuvent utiliser les données pour élaborer leurs stratégies de marketing numérique et de collecte de fonds.
- Les fournisseurs de services auprès du secteur sans but lucratif peuvent analyser les données pour connaître les tendances émergentes, et utiliser les données dans leur stratégie de marketing par contenu (billets de blogue, rapports, webinaires, etc.).
- Les médias peuvent utiliser les données dans des articles et des projets de recherche.

Comme source des données, mentionnez simplement « Projet de données ouvertes 2021 » et ajoutez un lien vers [**nptechforgood.com/open-data-project/projet-de-donnees-ouvertes**](http://www.nptechforgood.com/open-data-project/projet-de-donnees-ouvertes/)
